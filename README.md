# Development

### 1. Install xcode, node and yarn

```
xcode-select --install
brew install node yarn
```

### 2. Rustc and Cargo Package Manager

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### 3. Install dependecies

```
yarn install
```

### 4. Start dev server

```
yarn tauri dev
```
